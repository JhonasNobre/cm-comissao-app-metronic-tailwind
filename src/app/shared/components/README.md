# Shared Components

Componentes reutilizáveis usados em múltiplas features.

## Quando adicionar aqui?

Apenas componentes que são:
- 100% genéricos
- Usados em 2+ features diferentes
- Sem lógica de negócio específica

## Exemplos

- Botões customizados
- Modais genéricos
- Breadcrumbs
- Cards reutilizáveis
